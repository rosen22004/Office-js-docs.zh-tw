
# <a name="description-element"></a>Description 項目
將您 Office 增益集的描述指定為不超過 250 個字元的字串。

 **增益集類型︰**內容、工作窗格、郵件


## <a name="syntax:"></a>語法：


```XML
<Description DefaultValue="string ">
```
