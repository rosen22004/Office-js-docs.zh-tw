
# <a name="defaultlocale-element"></a>DefaultLocale 項目
指定增益集中字串使用之地區設定的預設文化特性名稱。

 **增益集類型︰**內容、工作窗格、郵件


## <a name="syntax:"></a>語法：


```XML
<DefaultLocale>string </DefaultLocale>
```


## <a name="remarks"></a>備註

以 BCP 47 `language` 標記格式來指定值，例如 en-US。
