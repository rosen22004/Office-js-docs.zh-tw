
# <a name="displayname-element"></a>DisplayName 項目
將您 Office 增益集的名稱指定為多達 125 個字元的字串。

 **增益集類型︰**內容、工作窗格、郵件


## <a name="syntax:"></a>語法：


```XML
<DisplayName DefaultValue="string ">
```
