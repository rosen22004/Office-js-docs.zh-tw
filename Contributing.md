## 常見問題集

### 我要如何取得 GitHub 帳戶？

填妥[加入 GitHub](https://github.com/join) 的表單以開啟免費 GitHub 帳戶。 

### 何處可以取得參與者的授權合約？ 

您會自動收到通知，提示您必須簽署參與者授權合約 (CLA) (如果您的提取要求需要 CLA)。 

身為社群成員，**您必須先簽署參與者授權合約 (CLA) 才可以參與此專案的大型提交**。 您只需要完成並提交該文件一次。 仔細檢閱文件。 您可能必須要求您的員工簽署此文件。
